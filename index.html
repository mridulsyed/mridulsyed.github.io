Thesis
Title: Convolutional Neural Network and Long Short-Term Memory based Human Activity Recognition System
Supervisor: Barshon Sen, Assistant Professor, CSE, RUET
Contribution:
Developed a novel 4-layer hybrid model combining Convolutional Neural Network (CNN) and Long Short-Term
Memory (LSTM) for efficient human activity detection using smartphone and smartwatch sensors.
Conducted experiments on UCI-HAR dataset, utilizing Bayesian Optimization for hyperparameter tuning.
Achieved an average accuracy increase of 2.16% with 10-fold cross-validation and Overlapping Window (OW) sample
generation, and 2.89% with 10-fold cross-validation and Non-overlapping Window (NOW).
Improved accuracy by 3.45% with Leave-one-subject-out (LOSO) validation and OW, and by 1.36% with LOSO and
NOW.
